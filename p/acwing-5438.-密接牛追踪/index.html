AcWing 5438. 密接牛追踪

AcWing 5438. 密接牛追踪

2025-03-27

题目：

农夫约翰有 NN 头奶牛排成一排，从左到右依次编号为 1∼N1∼N。

不幸的是，有一种传染病正在蔓延。

最开始时，只有一部分奶牛受到感染。

每经过一个晚上，受感染的牛就会将病毒传染给它左右两侧的牛（如果有的话）。

一旦奶牛被感染，它就会一直被感染，无法自愈。

给定一个经过若干个夜晚后的奶牛的整体状态，其中哪些奶牛已经被感染，哪些奶牛尚未被感染统统已知。

请你计算，最开始时就受到感染的奶牛的最小可能数量。

输入格式

第一行包含整数 NN。

第二行包含一个长度为 NN 的 0101 序列，用来表示给定的奶牛的整体状态，其中第 ii 个字符如果是 11 则表示第 ii 头奶牛已经被感染，如果是 00 则表示第 ii 头奶牛尚未被感染。

输出格式

一个整数，表示最开始时就受到感染的奶牛的最小可能数量。

数据范围

1≤N≤3×1051≤N≤3×105

输入样例1：

1
2

5
11111

输出样例1：

1

1

样例1解释

初始时，任意一头奶牛被感染，一定天数后都可以使得所有奶牛被感染。

输入样例2：

1
2

6
011101

输出样例2：

1

4

样例2解释

唯一一种可能是给定状态是没有经过任何夜晚时所有奶牛的状态，所以输入中被感染的 44 头奶牛都在最开始时就受到感染。

博主解题思路：

解题思路**

题目要求我们推测最初感染的奶牛最少可能有多少头，给定最终状态，我们可以通过 分段分析感染区域 来解决问题。

  1. 识别连续感染区域**

我们需要遍历字符串，找到 连续的 ‘1’ 段（即奶牛的感染区间）：

  • 遍历 s，找到 连续的 ‘1’ 段，记录其长度 c。
  • 记录所有 c 的值，形成 cnt 列表。
  1. 计算最优初始感染点**

在一个连续的 c 长度的感染段内，至少需要多少个初始感染点？

  • 假设每个初始感染点的传播范围为 r（左右扩展 r），那么：

    • 该感染点最多能覆盖 2r + 1 头奶牛。

  • 在每个 c 内，我们需要至少：

  • $$ ⌈c(2r+1)⌉\text{至少} \quad \left\lceil \frac{c}{(2r+1)} \right\rceil至少⌈(2r+1)c⌉头初始感染的奶牛。 $$
  1. 确定最优 r**

  • 对于每个感染段 c：

    • 如果 c 在 首尾位置（即 i==0 或 j==n），那么 r = c - 1。

  • $$ **否则** 选择 r = \frac{c - 1}{2}，这样能最优隔开初始感染点。 $$

  • 最终答案是最小 r 使得所有段的初始感染数最小。

  1. 计算最终结果**

  • 对于所有 c，我们计算：

  • $$ ∑(c+2r(2r+1))\sum \left( \frac{c + 2r}{(2r + 1)} \right)∑((2r+1)c+2r) $$

这个和就是最少的初始感染奶牛数。

  1. 时间复杂度**

  • 遍历一次 O(N) 找 1 段。

  • 计算 r 并统计 ans 也是 O(N)。

  • 总时间复杂度 O(N)，适用于

  • $$ N≤3×105N \leq 3 \times 10^5N≤3×105。 $$

import java.util.*;
import java.io.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int n = Integer.parseInt(br.readLine().trim()); // 读取奶牛数量
        String s = br.readLine().trim(); // 读取奶牛感染状态字符串

        List<Integer> cnt = new ArrayList<>(); // 存储连续的'1'的段长度
        int r = n; // 记录最优分割间隔

        // 遍历整个字符串
        for (int i = 0; i < n; i++) {
            if (s.charAt(i) == '0') continue; // 跳过未感染的奶牛
            int j = i + 1;
            while (j < n && s.charAt(j) == '1') j++; // 找到连续 '1' 的区间
            int c = j - i; // 计算当前段长度
            int d = (c - 1) / 2; // 计算能隔开的最小步长
            if (i == 0 || j == n) d = c - 1; // 如果在首尾，则直接取 c - 1
            r = Math.min(r, d); // 取最小值
            cnt.add(c); // 记录该段长度
            i = j; // 跳到下一个未访问的位置
        }

        int res = 0;
        for (int c : cnt) {
            res += (c + r * 2) / (r * 2 + 1); // 计算最少的初始感染奶牛数量
        }

        System.out.println(res); // 输出结果
    }
}
