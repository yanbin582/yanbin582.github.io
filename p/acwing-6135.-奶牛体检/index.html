AcWing 6135. 奶牛体检

2025-03-27

题目：

农夫约翰的 NN 头奶牛站成一行，奶牛 11 在队伍的最前面，奶牛 NN 在队伍的最后面。

农夫约翰的奶牛也有许多不同的品种。

他用从 11 到 NN 的整数来表示每一品种。

队伍从前到后第 ii 头奶牛的品种是 aiai。

农夫约翰正在带他的奶牛们去当地的奶牛医院进行体检。

然而，奶牛兽医非常挑剔，仅愿意当队伍中第 ii 头奶牛为品种 bibi 时对其进行体检。

农夫约翰很懒惰，不想完全重新排列他的奶牛。

他将执行以下操作恰好一次。

  • 选择两个整数 ll 和 rr，使得 1≤l≤r≤N1≤l≤r≤N。反转队伍中第 ll 头奶牛到第 rr 头奶牛之间的奶牛的顺序。

农夫约翰想要衡量这种方法有多少效果。

对于每一个 c=0…Nc=0…N，请帮助农夫约翰求出使得恰好 cc 头奶牛被检查的不同操作 (l,r)(l,r) 的数量。

两种操作 (l1,r1)(l1,r1) 和 (l2,r2)(l2,r2) 不同，如果 l1≠l2l1≠l2 或者 r1≠r2r1≠r2。

输入格式

输入的第一行包含 NN。

第二行包含 a1,a2,…,aNa1,a2,…,aN。

第三行包含 b1,b2,…,bNb1,b2,…,bN。

输出格式

输出 N+1N+1 行，第 ii 行包含使得 i−1i−1 头奶牛被检查的不同操作 (l,r)(l,r) 的数量。

样例1解释

如果农夫约翰选择 (l=1,r=1)(l=1,r=1)，(l=2,r=2)(l=2,r=2) 或 (l=3,r=3)(l=3,r=3)，则没有奶牛将会被检查。

注意这些操作并没有改变奶牛的位置。

以下操作会导致一头奶牛被检查。

  • (l=1,r=2)(l=1,r=2)：农夫约翰反转第一头和第二头奶牛的顺序，因此新队伍中每头奶牛的品种将为 [3,1,2][3,1,2]。第一头奶牛将会被检查。
  • (l=2,r=3)(l=2,r=3)：农夫约翰反转第二头和第三头奶牛的顺序，因此新队伍中每头奶牛的品种将为 [1,2,3][1,2,3]。第二头奶牛将会被检查。

博主解题思路：

解题思路

题目要求 反转区间 [l,r][l, r][l,r] 并计算 恰好 c 头奶牛被检查 的不同操作数量。 我们可以通过遍历所有可能的区间 [l,r][l, r][l,r] 来统计每种 c 值的出现次数。

  1. 初始匹配计数**

首先，计算 在没有反转任何元素的情况下，有多少头奶牛是匹配的：

  • cnt 记录当前匹配的奶牛数量，即 cnt = sum(a[i] == b[i])。
  1. 枚举所有反转区间**

遍历所有可能的 反转区间 [l,r][l, r][l,r]：

  • sum 代表在 反转 [l,r][l, r][l,r] 之前，匹配的奶牛数量（即 cnt）。
  • 逐步计算反转后匹配情况的变化：
    • 原本匹配的奶牛 若被反转到不匹配，sum--。
    • 原本不匹配的奶牛 若被反转后匹配，sum++。
  • ans[sum]++ 统计该 sum 值对应的操作数。
  1. 具体实现逻辑**

（1）枚举所有可能的区间**

  • 枚举所有可能的 区间起点 l。
  • 枚举所有可能的 区间长度 j=0,1，即：
    • 反转 奇数个元素（单中心）。
    • 反转 偶数个元素（双中心）。

（2）检查反转后的匹配情况**

  • 令 l 向左扩展，r 向右扩展，逐步统计匹配变化：
    • 如果 a[l] == b[l]，但反转后变得不匹配，则 sum--。
    • 如果 a[l] != b[l]，但反转后变得匹配，则 sum++。
  • 统计 不同 sum 值对应的操作次数。
  1. 时间复杂度分析**
  • 遍历所有可能的 (l, r)，时间复杂度为 O(N2)O(N^2)O(N2)，由于 N≤7500N \leq 7500N≤7500，可以接受。

import java.util.*;

public class Main {
    static final int N = 7510;
    static int n;
    static int[] a = new int[N];
    static int[] b = new int[N];
    static int[] ans = new int[N];
    
public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);

    n = scanner.nextInt();
    for (int i = 1; i <= n; i++) a[i] = scanner.nextInt();
    for (int i = 1; i <= n; i++) b[i] = scanner.nextInt();

    int cnt = 0;
    for (int i = 1; i <= n; i++) {
        if (a[i] == b[i]) cnt++;
    }

    for (int i = 1; i <= n; i++) {
        for (int j = 0; j < 2; j++) {
            int sum = cnt;
            for (int l = i, r = i + j; l >= 1 && r <= n; l--, r++) {
                if (a[l] == b[l]) sum--;
                if (a[r] == b[r]) sum--;
                if (a[l] == b[r]) sum++;
                if (a[r] == b[l]) sum++;
                ans[sum]++;
            }
        }
    }

    for (int i = 0; i <= n; i++) {
        System.out.println(ans[i]);
    }

    scanner.close();
}
}
