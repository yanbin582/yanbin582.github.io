AcWing 6134. 哞叫时间II

AcWing 6134. 哞叫时间II

2025-03-26

题目：

农夫约翰正在试图向埃尔茜描述他最喜欢的 USACO 竞赛，但她很难理解为什么他这么喜欢它。

他说「竞赛中我最喜欢的部分是贝茜说『现在是哞哞时间』并在整个竞赛中一直哞哞叫」。

埃尔茜仍然不理解，所以农夫约翰将竞赛以文本文件形式下载，并试图解释他的意思。

竞赛被定义为一个包含 NN 个整数的数组 a1,a2,…,aNa1,a2,…,aN。

农夫约翰定义哞叫为一个包含三个整数的数组，其中第二个整数等于第三个整数，但不等于第一个整数。

一种哞叫被称为在竞赛中发生，如果可以从数组中移除整数，直到只剩下这一哞叫。

由于贝茜据称「在整个竞赛中一直哞哞叫」，请帮助埃尔茜计算竞赛中发生的不同哞叫的数量！

两种哞叫是不同的，如果它们并非由相同的整数以相同的顺序组成。

输入格式

输入的第一行包含 NN。

第二行包含 NN 个空格分隔的整数 a1,a2,…,aNa1,a2,…,aN。

输出格式

输出竞赛中发生的不同哞叫的数量。

注意这个问题涉及到的整数可能需要使用 64 位整数型（例如，Java 中的 “long”，C/C++ 中的 “long long”）。

数据范围

1≤N≤1061≤N≤106, 1≤ai≤N1≤ai≤N

输入样例：

1
2

6
1 2 3 4 4 4

输出样例：

1

3

样例解释

竞赛包含三种不同的哞叫：1 4 4，2 4 4 和 3 4 4。

博主解题思路：

以下是针对该问题的解题思路：

问题理解

本题要求在一个包含 N 个整数的数组中，找出不同的 “哞叫” 数量。“哞叫” 是一个包含三个整数的数组，其中第二个整数等于第三个整数，但不等于第一个整数，并且这种 “哞叫” 可以通过从原数组中移除部分整数得到。

整体思路

为了计算不同的 “哞叫” 数量，我们需要找出所有可能的由三个整数组成的序列，满足第二个和第三个整数相同且与第一个整数不同。可以通过遍历数组，统计每个数字的左右两侧出现情况，从而找出所有符合条件的 “哞叫”。

具体步骤

1.初始化数据结构

  • 定义数组 w 用于存储输入的竞赛数组。
  • 定义数组 l 和 r 分别用于记录每个数字在左侧和右侧出现的次数。
  • 定义变量 cnt 用于统计当前不同数字的数量。
  • 定义变量 res 用于存储最终的不同 “哞叫” 数量，使用 Long 类型以处理可能较大的结果。
  1. 读取输入并统计唯一值
  • 读取输入的整数 n，表示竞赛数组的长度。
  • 读取整行输入并按空格拆分，将每个整数存储到数组 w 中。
  • 在存储过程中，使用 l 数组统计每个数字的出现次数。当某个数字第一次出现时，cnt 加 1，表示不同数字的数量增加。
  1. 遍历数组并计算 “哞叫” 数量
  • 从数组的最后一个元素开始向前遍历：
    • 对于当前元素 x，更新 r[x] 加 1，表示该数字在右侧的出现次数增加；同时更新 l[x] 减 1，表示该数字在左侧的出现次数减少。
    • 如果 l[x] 变为 0，说明该数字在左侧不再出现，cnt 减 1。
    • 当 r[x] 等于 2 时，说明当前找到了一个可能的 “哞叫” 的后两个相同数字。此时，不同的 “哞叫” 数量增加 cnt，因为可以选择左侧的任意一个不同数字作为 “哞叫” 的第一个数字。
    • 但是，如果 l[x] 大于 0，说明左侧还有与 x 相同的数字，而 “哞叫” 要求第一个数字与后两个不同，所以需要将 res 减 1。
  1. 输出结果
  • 将最终的不同 “哞叫” 数量 res 转换为字符串并写入输出流，最后关闭输入输出流。

复杂度分析

  • 时间复杂度：，其中 是数组的长度。因为只需要遍历数组一次。
  • 空间复杂度：，主要用于存储数组 w、l 和 r，其中 是输入数据范围的上限。

public static void main(String[] args) throws IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

    int[] w = new int[N];
    int[] l = new int[N];
    int[] r = new int[N];
    int cnt = 0; // 统计不同数的数量

    int n = Integer.parseInt(br.readLine());  // 读取 n

    // **修改：读取整行，然后拆分**
    String[] inputs = br.readLine().split(" ");
    for (int i = 1; i <= n; i++) {
        int x = Integer.parseInt(inputs[i - 1]); // **改为按空格拆分后的数组**
        w[i] = x;
        if (++l[x] == 1) cnt++; // 统计唯一值
    }

    Long res = 0L;
    for (int i = n; i > 0; i--) {
        int x = w[i];
        r[x]++;
        l[x]--;
        if (l[x] == 0) cnt--;
        if (r[x] == 2) {
            res += cnt;
            if (l[x] > 0) res--;
        }
    }

    bw.write(String.valueOf(res));
    bw.newLine();
    br.close();
    bw.close();
}

}
